# ParkBuddy Web

ParkBuddy Android 앱에서 진행했던 기능을 React 웹앱 형태로 옮긴 MVP 버전입니다.

## 포함 기능

- 회원 등록/수정/삭제
- 연락처 자동 하이픈 처리
- 필수항목 검사
- CSV 샘플 다운로드
- CSV 회원 일괄등록
- 라운딩 생성
- 도/광역시 선택 후 파크골프장 선택
- 참가자 선택
- 조장 후보 선택
- 조편성 방식 선택
- 개인전 실력 균형 조편성
- 직전 라운딩 같은 조 중복 최소화
- 포섬/포볼 2인 팀전 편성
- 홀별 점수 입력
- 스트로크 플레이 / 신페리오 순위 계산
- 라운딩 기록 저장 및 목록 보기

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 Vite가 안내하는 주소를 열면 됩니다.

## 배포

`main` 브랜치에 푸시하면 GitHub Actions가 빌드 후 GitHub Pages로 배포합니다.

예상 배포 주소:

```text
https://joon7jihoo-ctrl.github.io/parkbuddy-webapp/
```

## 현재 저장 방식

이 버전은 MVP 웹앱으로, 데이터를 브라우저 화면 상태에 저장합니다.
새로고침하면 데이터가 초기화됩니다.

다음 단계에서는 localStorage 또는 Supabase/Firebase 같은 백엔드를 붙여 영구 저장할 수 있습니다.

## 조편성 기준

참가자별 실력 점수는 평균 타수, 참가 횟수, 실력 등급을 함께 사용합니다.

- 평균 타수: 50%
- 참가 횟수: 30%
- 실력 등급: 20%

평균 타수나 참가 횟수 기록이 없는 회원도 기본값으로 안전하게 계산합니다.
스트로크 플레이, 신페리오, 매치 플레이는 개인전 조편성으로 처리하고, 포섬과 포볼은 2인 팀을 먼저 만든 뒤 2개 팀을 한 조로 배치합니다.
