# ParkBuddy Web

ParkBuddy Android 앱에서 진행했던 기능을 React 웹앱 형태로 옮긴 MVP 버전입니다.

## 포함 기능

- 회원 등록/수정/삭제
- 연락처 자동 하이픈 처리
- 필수항목 검사
- CSV 샘플 다운로드
- CSV 회원 일괄등록
- 라운딩 생성
- 도/광역시 선택 후 파크골프장 선택
- 참가자 선택
- 조장 후보 선택
- 조편성
- 홀별 점수 입력
- 스트로크 플레이 / 신페리오 순위 계산
- 라운딩 기록 저장 및 목록 보기

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 Vite가 안내하는 주소를 열면 됩니다.

## 현재 저장 방식

이 버전은 MVP 웹앱으로, 데이터를 브라우저 화면 상태에 저장합니다.
새로고침하면 데이터가 초기화됩니다.

다음 단계에서는 localStorage 또는 Supabase/Firebase 같은 백엔드를 붙여 영구 저장할 수 있습니다.
